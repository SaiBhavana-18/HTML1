b tag

The b HTML element is used to draw the reader's attention to the element's contents, which are not otherwise granted special importance. This was formerly known as the Boldface element, and most browsers still draw the text in boldface

i tag

The i HTML element is used to display the content in italic

u tag

The u HTML element is used to highlight the content by underlining it

select tag

The select HTML element is used to create a drop down list

button tag

The button tag defines a clickable button

hr tag

the hr tag defines a thematic break in an HTML page(e.g. a shift of topic)

br tag

The br tag inserts a single line break. The br tag is useful for writing addresses or poems. The br tag is an empty tag which means that it has no end tag.
